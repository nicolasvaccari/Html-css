Nesta frase temos um termo em destaque usando a tag STRONG(semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I(não semântica)

Nesta frase temos um termo em Ênfase usando a tag EM(semântica)

Conclusão

    Modificar um ou mais elemento

    Para modificar somente um elemento, é preciso usar a tag id
    Para modificar mais de um elemento, é preciso usar a tag class


    exemplo: <h1 id="principal">
    <h1 class="principal2">

    Para o Css, o id significa #.(h1#principal)

    Para o Css, o class significa . (."básico")
